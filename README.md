## ğŸ¨ Customization

### Changing Colors
Main colors are defined in CSS variables at the beginning of the `styles.css` file:
- `#000000` - Deep Black (primary text, logo)
- `#DAA520` - Golden Yellow (grain, harvest symbolism)
- `#006400` - Emerald Green (nature, sustainability)
- `#2F4F4F` - Steel Gray (industry, logistics)

### Brand Colors
The website uses LOCKWOOD TRADING LIMITED's official brand colors:
- **Deep Black (#000000)**: Strength, reliability, prestige
- **Golden Yellow (#DAA520)**: Grain, harvest, sun symbolism
- **Emerald Green (#006400)**: Nature, fields, sustainability
- **Steel Gray (#2F4F4F)**: Industry, ships, logistics

### Changing Content
1. **Company Name** - Change all mentions of "LOCKWOOD TRADING LIMITED" in the HTML file
2. **Contact Information** - Update data in the contact section
3. **Services** - Modify service cards to fit your needs
4. **Statistics** - Update numbers in the "About Company" section

### Adding New Sections
1. Create a new section in HTML
2. Add a link to the navigation
3. Apply styles in CSS
4. Add JavaScript functionality if needed
